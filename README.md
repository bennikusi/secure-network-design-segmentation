# Secure Network Design & Segmentation

## Overview
Designed and implemented a secure small-business network architecture using VLAN segmentation, ACL enforcement, firewall policies, and inter-VLAN routing to strengthen internal network security and reduce unauthorized access risks.

## Security Objectives
- Segment departments using VLANs
- Restrict unauthorized communication
- Reduce lateral movement risks
- Enforce least-privilege access controls
- Improve internal network security

## Tools & Technologies
- Cisco Packet Tracer
- VLANs
- ACLs
- Inter-VLAN Routing
- Firewall Policies
- Network Segmentation

## Key Implementations
- Configured VLAN segmentation for different departments
- Applied ACL-based firewall policies
- Restricted unauthorized inter-VLAN traffic
- Implemented router-on-a-stick inter-VLAN routing
- Performed connectivity and access-control validation testing

## Validation & Testing
Testing was performed using ping validation between VLANs. Authorized communication succeeded while restricted traffic was successfully blocked according to ACL firewall policies.

## Security Impact
The implemented controls strengthened infrastructure security through network segmentation, least-privilege access enforcement, and reduced lateral movement risks.
